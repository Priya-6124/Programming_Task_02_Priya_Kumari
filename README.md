# Programming Task 02

## Objective

This task focuses on strengthening programming fundamentals through decision-making and logical problem-solving using Python programming.

The concepts covered in this task include:

* Variables
* User Input Handling
* Conditional Statements
* Comparison Operators
* Logical Operators
* Basic String Handling
* Problem Solving Techniques

---

## Programs Included

### 1. Even or Odd Checker (`even_odd.py`)

This program accepts a number from the user and determines whether the number is even or odd using the modulus (`%`) operator.

**Example Output:**

```
Enter a number: 15
15 is an Odd Number
```

---

### 2. Largest of Three Numbers (`largest_number.py`)

This program accepts three numbers as input and identifies the largest number using conditional statements.

**Example Output:**

```
Enter First Number: 10
Enter Second Number: 25
Enter Third Number: 18

Largest Number = 25
```

---

### 3. Student Grade Calculator (`grade_calculator.py`)

This program accepts marks from the user and assigns a grade according to predefined grading criteria.

| Marks    | Grade |
| -------- | ----- |
| 90 – 100 | A     |
| 80 – 89  | B     |
| 70 – 79  | C     |
| 60 – 69  | D     |
| Below 60 | F     |

**Example Output:**

```
Enter Marks: 85
Grade: B
```

---

### 4. Login Validation (`login_validation.py`)

This program validates a username and password entered by the user against predefined credentials.

**Default Credentials:**

* Username: admin
* Password: password123

**Example Output:**

```
Enter Username: admin
Enter Password: password123

Login Successful
```

---

### 5. Password Strength Checker (`password_checker.py`)

This program evaluates the strength of a password based on the following conditions:

* Minimum length of 8 characters
* At least one uppercase letter
* At least one numeric digit

**Strength Levels:**

* Weak Password
* Moderate Password
* Strong Password

**Example Output:**

```
Enter Password: Cyber123
Strong Password
```
